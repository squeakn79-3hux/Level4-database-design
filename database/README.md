# Database

This folder contains the MySQL database export and supporting documentation for the Bright Sparks Ltd Inventory Management Database System.

## Files

| File | Purpose |
|---|---|
| `BrightSparks_Database.sql` | Full MySQL export of the completed database. |
| `Schema.md` | Short overview of the database tables and fields. |
| `Relationships.md` | Summary of the table relationships. |
| `Tables.md` | Table-by-table breakdown. |

## Import

Use MySQL Workbench to import `BrightSparks_Database.sql`.

1. Open MySQL Workbench.
2. Go to **Server > Data Import**.
3. Select **Import from Self-Contained File**.
4. Choose `BrightSparks_Database.sql`.
5. Start the import.
