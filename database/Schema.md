# Schema

The database export contains the following tables:

| Table | Purpose |
|---|---|
| `category_table` | Stores product category information. |
| `inventory_table` | Tracks stock quantities, stock on order and quantities sold. |
| `product_table` | Stores product details and links each product to a supplier and category. |
| `sales_table` | Stores sales transaction information. |
| `sold_relationship_table` | Links sales transactions to the products sold. |
| `supplier_table` | Stores supplier contact and address information. |

The structure supports products, suppliers, inventory, sales and reporting for Bright Sparks Ltd.
